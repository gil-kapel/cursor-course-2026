# Lesson 1.3 — Terminal, Git, and safety nets

Hebrew-first lesson assets; this index is in English to match the course README.

**Scope:** Demystifying the terminal as the surface where **Cursor’s agent** proposes shell commands you approve; the "read before you run" reflex; Git as a safety net (**commit + restore** only in this lesson). No branches, no remotes, no merge.

## Syllabus topic

Easing "black screen" anxiety; opening the terminal in Cursor; the **agent in Cursor** showing commands for user approval; asking for explanations before Run; Git as "save game"; hands-on Git setup via chat; save-before-you-build habit.
