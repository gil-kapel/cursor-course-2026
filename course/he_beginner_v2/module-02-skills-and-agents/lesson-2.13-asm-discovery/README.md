## Syllabus topic

Practical **Agent Skill Manager (ASM)** workflow: `asm search`, evaluating candidates, `asm add skill`, `asm sync`, grouping with `asm create expertise`, optional `asm skill analyze … --cloud`.

## Prerequisites

- [Lesson 2.1](../lesson-2.1-skills-intro/README.md) and [2.2 skill anatomy](../lesson-2.12-skill-anatomy/README.md).

## Learning outcomes

- Run **`asm search`** with a concrete query and interpret results.
- Read **AGENT_SKILLS.md** scoring columns (lesson fit, authority, ecosystem, Cursor fit, overlap).
- Install a skill with **`asm add skill`** and materialize agent config with **`asm sync`**.
- Create an **expertise bundle** with `asm create expertise … --desc "…"`.
