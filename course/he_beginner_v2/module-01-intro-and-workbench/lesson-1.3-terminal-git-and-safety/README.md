# Lesson 1.3 — Terminal, Git, and safety nets

Hebrew-first lesson assets; this index is in English to match the course README.

**Scope:** Demystifying the terminal as the surface where **Claude Code** runs shell commands (per [official overview](https://code.claude.com/docs/en/overview)); the "read before you run" reflex; Git integration (officially Claude Code works with Git — this lesson teaches only **commit + restore** as a safety net). No branches, no remotes, no merge.

## Syllabus topic

Easing "black screen" anxiety; opening the terminal in Cursor; **Claude Code** showing commands for user approval; asking for explanations before Run; Git as "save game" aligned with product capabilities; hands-on Git setup via **Claude Code**; save-before-you-build habit.
