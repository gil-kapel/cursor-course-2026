## Syllabus topic

How to start any agent task with a **sharp prompt** and the right **context pack**. This lesson turns the whole module into a practical operating system: one task, one clean context, one clear expected output.

## Practical outcomes

- Build short starter prompts that define goal, inputs, output, constraints, and done criteria.
- Use `@` file attachment intentionally instead of dumping context.
- Keep context isolated between tasks so agents do not pollute one another.
- Save 2-3 reusable prompts as personal playbooks.
