## Syllabus topic

Understanding the difference between **Cursor Rules** (`.cursor/rules/*.mdc`) and **Agent Skills** (`SKILL.md`). Knowledge injection vs. workflow orchestration. Identifying where these assets live on disk and how they appear in Cursor settings.

## Practical outcomes

- Identify one active project rule and one installed skill in a real workspace.
- Explain the difference between a project default and an on-demand workflow.
- Ask Cursor to summarize the current Rules and Skills in the project.

## Verify

- You can point to `.cursor/rules` and `.cursor/skills` and explain what each one is for.
- The agent can summarize what it found without changing any files.
