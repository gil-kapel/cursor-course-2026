# Lesson 1.6 — Chat modes: Agent, Ask & Plan

Hebrew-first lesson assets; this index is in English to match the course README.

**Scope:** Deep dive into Cursor's four chat modes — Agent (the builder), Ask (the advisor), Plan (the architect), and Debug (the fixer). Builds on the brief Ask/Agent introduction in 1.4. Introduces Plan mode (Shift+Tab) as the "think before you build" workflow. Includes a decision framework and a combined Plan-to-Debug hands-on demo.

## Syllabus topic

Agent mode as primary builder (recap from 1.4); Ask mode for read-only exploration and understanding; Plan mode (Shift+Tab) for researching the codebase, generating a reviewable plan, and building only after approval — the "development manager" mindset; Debug mode for targeted troubleshooting; decision framework (simple → Agent, understanding → Ask, complex → Plan then Agent, broken → Debug); combined Plan-to-Debug hands-on demo. Bridges to lesson 1.7 (Cursor Settings).
