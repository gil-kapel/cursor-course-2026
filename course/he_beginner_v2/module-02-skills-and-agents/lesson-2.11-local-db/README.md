## Syllabus topic

Lightweight persistence during development: when SQLite vs JSON fits, schema-first thinking, and keeping local data disposable before cloud databases.

## Learning outcomes

- Choose between **SQLite** and **JSON** for early dev data and justify the tradeoff.
- Think **schema-first** and treat local data as **disposable** until you graduate to a hosted database.
