# קורס Cursor בעברית

## מטרת הקורס

להביא משתמש חדש או מפתח שמכיר כלי AI אבל חדש ב-Cursor, מרגע ההתקנה ועד עבודה עצמאית על פיצ'ר מלא עם תכנון, מימוש, דיבוג, עבודה עם worktrees ופריסה לענן.

## קהל יעד

- מתחילים שרוצים להבין איך לעבוד עם Cursor בצורה בטוחה ומעשית.
- מפתחים חדשים ל-Cursor שרוצים שיטת עבודה ברורה ולא רק "לזרוק פרומפטים".
- צוותים קטנים שרוצים להגדיר הקשר פרויקט, כללים, MCPs ותהליך עבודה עקבי.

## מבנה הקורס

- פורמט: 15 סרטונים
- אורך מומלץ: 5 עד 10 דקות לסרטון
- סגנון: דמו מעשי קודם, אחר כך הסבר קצר, ואז צ'קליסט פעולה

## סטנדרט שיעור

- לכל שיעור יש `Script Master`, `Prompt Asset`, `Recording Notes`, ו-`Action Item`.
- ה-`Scene List` משמש גם כבסיס לחלוקת קריינות ל-TTS.
- הנחיות הפקה וקריינות משותפות נמצאות ב-`course/he/docs/recording-and-elevenlabs-guide.md`.
- עקביות מונחים נשמרת לפי `course/he/docs/hebrew-tts-glossary.md`.

## סדר השיעורים

### 01. מבוא ותוצאה סופית

- מטרה: להסביר מה Cursor פותר, למי הקורס מיועד, ומה הסטודנט ידע לעשות בסוף.
- תוצאה לומדת: הסטודנט מבין את מסלול הקורס ואת צורת העבודה שנלמדת בו.
- דמו מרכזי: מעבר מהיר על סביבת Cursor ועל פרויקט שייבנה במהלך הקורס.

### 02. התקנה על macOS ו-Windows

- מטרה: לבצע התקנה ראשונה בצורה נקייה ולוודא שהכול עובד.
- תוצאה לומדת: הסטודנט מתקין את Cursor, פותח פרויקט ראשון ומוודא שהסביבה תקינה.
- דמו מרכזי: הורדה, התקנה, פתיחת תיקייה והרצת בדיקת sanity ראשונה.

### 03. פתיחת חשבון ובחירת מנוי

- מטרה: להבין את מסלולי המנוי ולבחור מסלול מתאים לפי שימוש.
- תוצאה לומדת: הסטודנט יודע מתי חבילת חינם מספיקה ומתי כדאי לשדרג ל-Pro.
- דמו מרכזי: מסך הרשמה, בחירת תוכנית, הסבר על שיקולי תקציב ושימוש.

### 04. אסטרטגיית מודלים ותמחור

- מטרה: להסביר איך בוחרים מודל לפי משימה ולא לפי הייפ.
- תוצאה לומדת: הסטודנט יודע להתאים מודל לתכנון, מימוש, דיבוג וריוויו, ולהבין איך tokens, context וגודל תשובה משפיעים על עלות.
- דמו מרכזי: השוואת משימות קצרות מול משימות עומק והשלכתן על עלות, מהירות וכמות context.

### 04b. ניהול הקשר: פחות טוקנים, יותר דיוק

- מטרה: ללמד איך לנהל context לכל משימה כך שהמודל יקבל רק את מה שרלוונטי.
- תוצאה לומדת: הסטודנט יודע לצמצם scope, לסכם thread ארוך, לבקש output ממוקד, ולהוריד עלות ורעש.
- דמו מרכזי: השוואה בין prompt עמוס עם הרבה context לבין prompt ממוקד עם handoff קצר וברור.

### 05. מודים ואייג'נטים בסיסיים

- מטרה: להבין מתי להשתמש ב-Ask, Plan, Agent ו-Debug.
- תוצאה לומדת: הסטודנט יודע לבחור mode נכון לכל שלב במשימה.
- דמו מרכזי: אותה משימה מוצגת בארבעה modes שונים.

### 06. הקשר פרויקט: rules, skills, memory, MCPs ותוספים

- מטרה: לבנות הקשר פרויקט טוב שמכוון את Cursor מההתחלה.
- תוצאה לומדת: הסטודנט יודע להגדיר כללי עבודה, קבצי הקשר, skill מתאים, MCPs מומלצים ותוספים בסיסיים.
- דמו מרכזי: יצירת structure מומלץ לפרויקט והפעלת MCPs רלוונטיים.

### 07. לופ עבודה לבניית פיצ'ר מאפס

- מטרה: להראות תהליך עקבי של plan -> implement -> validate -> debug.
- תוצאה לומדת: הסטודנט יודע לפרק פיצ'ר לפרוסות קטנות ולעבוד בצעדים בטוחים.
- דמו מרכזי: בניית פיצ'ר קטן מהגדרה ועד ולידציה.

### 08. דיבוג והתאוששות מתקלות

- מטרה: ללמד איך לחזור לשליטה כש-Cursor או הקוד "בורחים".
- תוצאה לומדת: הסטודנט יודע לצמצם בעיה, לנסח prompt טוב לדיבוג, ולהגדיר תיקון קטן ובטוח.
- דמו מרכזי: תיקון רגרסיה עם מסקנה, היפותזות ואימות.

### 09. Git ו-Worktrees

- מטרה: להראות איך עובדים במקביל על פיצ'רים ותיקונים בלי ללכלך את הענף הראשי.
- תוצאה לומדת: הסטודנט יודע לפתוח worktree, לחבר branch נכון, ולנקות בסוף.
- דמו מרכזי: שני worktrees במקביל, אחד לפיצ'ר ואחד לבאג.

### 10. איכות קוד ושומרי סף

- מטרה: להראות איך לא לתת ל-AI להכניס קוד מסוכן או לא קריא.
- תוצאה לומדת: הסטודנט יודע לבקש review, לזהות סיכונים, ולהריץ בדיקות לפני commit.
- דמו מרכזי: ריוויו ממוקד סיכונים לפני merge.

### 11. עבודה בצוות והעברת משימות

- מטרה: לייצר handoff ברור בין אנשים שעובדים עם Cursor.
- תוצאה לומדת: הסטודנט יודע לנסח handoff, לשתף הקשר, ולהעביר next steps.
- דמו מרכזי: יצירת חבילת handoff למפתח נוסף.

### 12. פריסה לענן: Vercel ואלטרנטיבות

- מטרה: לחבר את מה שנבנה לפריסה אמיתית ולהסביר איך בוחרים ספק ענן.
- תוצאה לומדת: הסטודנט יודע לבצע deploy בסיסי, להגדיר env vars ולהבחין בין preview ל-production.
- דמו מרכזי: deploy ל-Vercel והשוואה ל-Render, Netlify, Railway או Fly.io.

### 13. פרויקט מסכם

- מטרה: לחבר את כל תהליך העבודה לפרויקט קטן אחד.
- תוצאה לומדת: הסטודנט בונה יכולת עבודה מלאה עם Cursor מקצה לקצה.
- דמו מרכזי: mini project עם checkpoints לאורך הדרך.

### 14. פרסום, יצוא ולוקליזציה

- מטרה: להכין את הקורס לשפות נוספות בלי לבנות הכול מחדש.
- תוצאה לומדת: הסטודנט מבין איך מפרידים script master, captions וגלוסריום.
- דמו מרכזי: מעבר מהיר על pipeline של Hebrew-first ואז localization.

- לבנות פיצ'ר דרך לופ קבוע.
- לעבוד עם worktrees בבטחה.
- לפרוס פרויקט קטן לסביבת ענן.
- להשתמש בפרומפטים מוכנים לכל שלב.
