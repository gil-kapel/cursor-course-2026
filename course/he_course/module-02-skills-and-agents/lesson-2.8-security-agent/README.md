## Scope

Using Plan mode to combine all existing documents into an ordered implementation plan with slices. Slice 1: task list + empty state + CTA. Slice 2: add task + validation. Slice 3: update status + delete.

## Verify

- `docs/implementation-plan.md` exists with clear slice order.
- Each slice has: goal, expected files, verification, out of scope.
- Slice 1 is concrete enough to implement without guessing.
