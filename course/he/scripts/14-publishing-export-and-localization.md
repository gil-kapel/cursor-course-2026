# Script Master: 14 Publishing, Export, and Localization

- Title: פרסום, יצוא ולוקליזציה

## Lesson Goal

להראות איך מכינים את הקורס לפרסום ולשפות נוספות בלי לבנות הכול מחדש: הפרדה בין script master, prompts, captions וגלוסריום, וזרימת עבודה מסודרת לייצוא וללוקליזציה.

## Prompt Asset

`Help me prepare this lesson for publishing and localization. Treat the script master as the source of truth, identify which assets should be derived from it, preserve a consistent glossary for key Cursor terms, and produce a checklist for captions, prompts, export structure, and multilingual adaptation.`
