## Learning outcomes

- Name the **two ways** a Skill adds value: **knowledge the model lacks** vs **structured multi-step workflow**.
- Tell a **Cursor project Rule** (conventions, stack, globs / always-on) apart from a **Skill** (reusable task playbook in `SKILL.md`).
- Locate **project rules** in Cursor Settings and **`.cursor/rules` / `.cursor/skills`** in the repo.
- Know that **ASM installation** and **skill discovery** are covered in app lesson **2.3**, and **trigger / quality** details in **2.2**.
