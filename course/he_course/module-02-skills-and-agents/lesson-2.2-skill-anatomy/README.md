## Syllabus topic

Understanding how the AI decides to use a skill. The importance of the YAML frontmatter (triggers), progressive disclosure, and the difference between a high-quality `SKILL.md` and a vague one. Deconstructing a real skill into its core components: triggers, workflow, and output.
