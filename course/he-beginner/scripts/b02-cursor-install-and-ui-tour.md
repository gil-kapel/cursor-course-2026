- Target duration: 09:00

## Lesson Goal

התקנה ראשונה נקייה והבנת הממשק דרך אנלוגיות: ארון קבצים, שולחן עבודה, ערוץ פקודות לסוכן, מסוף.

## Examples (דוגמאות קונקרטיות)

### מה נראה על המסך בתחילת העבודה

- `Open Folder` על תיקייה ריקה בשם `cursor-practice`.
- קובץ ראשון כמו `notes.md` או `hello.txt`.
- מסוף משולב פתוח בתוך אותו חלון, לא חלון נפרד.

### אנלוגיות שיעזרו למתחילים

- Explorer = ארון קבצים.
- Editor = שולחן העבודה שעליו מניחים את הדף הפתוח.
- AI chat = המקום שבו נותנים הוראות ושואלים שאלות.
- Terminal = אזור שבו רצות פקודות כשצריך.

### טעויות נפוצות שכדאי להדגים

- לפתוח רק קובץ בודד במקום תיקייה שלמה.
- לעבוד מחוץ לתיקיית הפרויקט בלי לשים לב.
- לחפש הכול בהגדרות במקום להשתמש ב-Command Palette.

## User tasks — pause and verify (עצירות בכיתה)

**הנחיה למנחה:** לבקש מהלומדים לבצע על המחשב שלהם ולא רק לצפות.

### משימה 1 — התקנה פתוחה (אחרי S02)

- **עצירה:** אחרי ההורדה וההתקנה.
- **מה לעשות:** לפתוח את Cursor בהצלחה על המחשב שלכם.
- **איך לוודא הצלחה:** חלון Cursor פתוח, בלי להישאר על קובץ התקנה או מסך הורדה בלבד.

### משימה 2 — פתיחת תיקייה נכונה (אחרי S03)

- **עצירה:** אחרי `Open Folder`.
- **מה לעשות:** ליצור תיקייה בשם `cursor-practice` ולפתוח אותה ב-Cursor.
- **איך לוודא הצלחה:** שם התיקייה מופיע ב-Explorer, והיא ריקה או כמעט ריקה.

### משימה 3 — לזהות את האזורים ולשמור קובץ ראשון (אחרי S04-S06)

- **עצירה:** אחרי הסיור בממשק, המסוף, ויצירת הקובץ הראשון.
- **מה לעשות:** לזהות את ארבעת האזורים המרכזיים, ואז ליצור `notes.md` או `hello.txt`, לכתוב משפט אחד, ולשמור.
- **איך לוודא הצלחה:** אתם יודעים להגיד במשפט אחד מה תפקיד כל אזור, והקובץ מופיע ב-Explorer עם התוכן ששמרתם.
