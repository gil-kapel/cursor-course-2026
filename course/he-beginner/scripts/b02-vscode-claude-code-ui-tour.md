## Lesson Goal

התקנת **Cursor** (אם עדיין אין), התקנת ההרחבה הרשמית **Claude Code for VS Code** (Anthropic) מתוך ממשק ההרחבות של Cursor, והבנת הממשק: Explorer, עורך, פאנל Claude Code, מסוף משולב. כל המסלול הבסיסי מניח **Cursor + הרחבה זו**; מי שמעדיף יכול לבצע את אותן צעדים ב-**VS Code** נקי.

## Examples (דוגמאות קונקרטיות)

### מה נראה על המסך בתחילת העבודה

- Cursor פתוח עם `Open Folder` על תיקייה ריקה בשם `claude-cursor-practice`.
- ההרחבה **Claude Code** (מפרסם Anthropic) מותקנת; פאנל/כרטיסיית Claude Code זמינים.
- קובץ ראשון כמו `notes.md` או `hello.txt`.
- מסוף משולב (`Terminal → New Terminal`) באותו חלון.

### אנלוגיות שיעזרו למתחילים

- Explorer = ארון קבצים.
- Editor = שולחן העבודה.
- פאנל Claude Code = שיחה עם הסוכן, תוכניות, והצעות שינוי (כולל diff לפני קבלה).
- Terminal = פקודות; לפי התיעוד, לעיתים גם זרימת CLI — בשלב בסיסי מספיק מסוף משולב.

### טעויות נפוצות שכדאי להדגים

- לפתוח רק קובץ בודד במקום תיקיית פרויקט שלמה.
- להתקין הרחבה לא רשמית במקום **Claude Code** של Anthropic מה-Marketplace.
- לחפש הכול בהגדרות במקום **Command Palette** (`Cmd+Shift+P` / `Ctrl+Shift+P`).

- **עצירה:** אחרי התקנת Cursor (אם נדרש) והתקנת Claude Code.
- **מה לעשות:** לפתוח Cursor ולוודא שההרחבה **Claude Code** (Anthropic) מופיעה כמותקנת.
- **איך לוודא הצלחה:** אפשר לפתוח את פאנל/כרטיסיית Claude Code או לראות את ההרחבה ברשימת Extensions.

### משימה 2 — פתיחת תיקייה נכונה (אחרי S03)

- **עצירה:** אחרי `Open Folder`.
- **מה לעשות:** ליצור תיקייה בשם `claude-cursor-practice` ולפתוח אותה ב-Cursor.
- **איך לוודא הצלחה:** שם התיקייה ב-Explorer, תיקייה ריקה או כמעט ריקה.

### משימה 3 — אזורים וקובץ ראשון (אחרי S04–S06)

- **עצירה:** אחרי סיור בממשק, מסוף, וקובץ ראשון.
- **מה לעשות:** לזהות Explorer, עורך, פאנל Claude Code, מסוף; ליצור `notes.md` או `hello.txt`, משפט אחד, שמירה.
- **איך לוודא הצלחה:** ארבעת האזורים ברורים; הקובץ נשמר ומופיע ב-Explorer.

### S01 | 00:00–00:40 | Hook — Cursor + הרחבה

- **מה על המסך:** לוגו Cursor + אייקון Spark / "Claude Code" בצד; רקע נקי. אין התקנה בשלב הזה.
- **פעולות:** פייד אין או זום קל על לוגו ההרחבה מתוך רשימת Extensions (צילום מראש).
- **קריינות:** "במסלול הבסיסי אנחנו יושבים על **Cursor** — עורך מבוסס VS Code — ומתקינים את ההרחבה הרשמית **Claude Code for VS Code** של Anthropic. כך מקבלים diff, פרויקט, ומסוף באותו חלון. מי שרוצה VS Code נקי יכול לחזור על אותם צעדים שם."
- **אם לא מספיק:** שקופית עם שני לוגואים + משפט; בלי צילום חי.

### S02 | 00:40–02:05 | התקנת Cursor והרחבה

- **מה על המסך:** (1) דף הורדה cursor.com — גלילה לכפתור הורדה; (2) חלון Cursor → סרגל צד **Extensions** (אייקון ריבועים); (3) שדה חיפוש `Claude Code` — תוצאה עם **Publisher: Anthropic**; (4) כפתור Install → Reload אם מופיע.
- **פעולות:** קצב איטי; עצירה 2 שניות על שורת המפרסם Anthropic כדי שהצופה יקרא.
- **קריינות:** "אם אין לכם Cursor — מורידים מהאתר הרשמי. ב-Extensions מחפשים **Claude Code**, מתקינים את זה שמפרסם **Anthropic**. אם ההרחבה לא מופיעה — Reload לחלון לפי מה שהמוצר מציע. דרישות גרסה — לפי [תיעוד ההרחבה](https://code.claude.com/docs/en/vs-code) בזמן הצילום."
- **עצירת כיתה — משימה 1.**
- **אם לא מספיק:** Cursor כבר מותקן — דלגו על האתר; הציגו רק חיפוש ההרחבה והתקנה. אין רשת — השתמשו בהקלטת מסך מוקדמת או שקופית עם חצים על אותם שלבים.

### S03 | 02:05–03:15 | Open Folder — לא קובץ בודד

- **מה על המסך:** תפריט **File → Open Folder** (או Welcome עם Open Folder); בוחרים תיקייה `claude-cursor-practice` ריקה; אחרי האישור — Explorer משמאל עם שם התיקייה בכותרת החלון.
- **פעולות:** הדגמה של **לא** לבחור "Open File" בלבד; אם בטעות נפתח קובץ בודד — סגרו והראו שוב Open Folder.
- **קריינות:** "Claude Code עובד על **תיקיית פרויקט**. יוצרים תיקייה ריקה, פותחים אותה ב-Cursor. כך הסוכן רואה את אותה תמונה שאתם רואים."
- **עצירת כיתה — משימה 2.**
- **אם לא מספיק:** תיקייה עם קובץ אחד בלבד — מחקו אותו בדמו או צרו תיקייה חדשה לפני הצילום.

### S04 | 03:15–04:25 | Explorer, עורך, Spark, diff

- **מה על המסך:** מסך מלא של Cursor: (1) Explorer; (2) עורך ריק או עם `notes.md`; (3) לחיצה על **Spark** — פאנל Claude Code נפתח; (4) גרירת גבול הפאנל **רחבה**; (5) אם אין diff חי — פתחו שינוי קטן ידני בקובץ והציגו את תצוגת השינוי של העורך או diff מההרחבה אם זמין.
- **פעולות:** מספר מספרים על המסך (1–4) בעריכה או בשקופית-שקופית; אל תגללו מהר בפאנל.
- **קריינות:** "משמאל — Explorer. במרכז — העורך. פאנל **Claude Code** נפתח דרך אייקון ה-**Spark**… **טיפ:** למתוח את סרגל הצד **רחב** כדי לראות שינויים inline…"
- **אם לא מספיק:** Spark לא נראה — Command Palette: חפשו "Claude" / "Claude Code" והפעילו את הפקודה שפותחת את הפאנל (עדכנו לפי גרסה). אין שינוי להדגים — הוסיפו שורה ל-`readme.txt` ידנית.

### S05 | 04:25–05:20 | מסוף משולב

- **מה על המסך:** **Terminal → New Terminal**; פרומפט בתוך תיקיית הפרויקט; אופציונלי: `pwd` / `cd` בלי סודות.
- **פעולות:** הדגישו שהמסוף **בתוך** אותו חלון Cursor, לא חלון נפרד של Terminal.app בלבד.
- **קריינות:** "המסוף המשולב: פקודות, לוגים, ולפעמים גם שימוש ב-CLI של Claude Code. בשיעור הבא נלמד לא ללחוץ Enter בלי להבין."
- **הכנה למשימה 3** (אמרו בקול: "עכשיו תזהו את ארבעת האזורים ותיצרו קובץ").

### S06 | 05:20–06:15 | קובץ ראשון ושמירה

- **מה על המסך:** לחיצה ימנית ב-Explorer → New File → `notes.md`; הקלדת משפט אחד; **Cmd+S / Ctrl+S**; נקודה לבנה נעלמת מעל הטאב (או סימן שמירה).
- **פעולות:** עצרו על הקובץ ב-Explorer אחרי השמירה כדי לראות שהוא מופיע.
- **קריינות:** "קובץ אחד, משפט אחד, שמירה. קודם ביטחון בעורך, אחר כך סוכן שמשנה הרבה קבצים."
- **עצירת כיתה — משימה 3.**
- **אם לא מספיק:** השתמשו ב-`hello.txt` במקום md; אם המקלדת בשידור בעייתית — הדגימו קובץ שמור מראש והסבירו "כך נשמר".

### S07 | 06:15–07:05 | Command Palette והגדרות

- **מה על המסך:** **Cmd+Shift+P / Ctrl+Shift+P** — רשימת פקודות; גלילה קצרה; **Settings** → חיפוש `Claude Code` תחת Extensions (אופציונלי).
- **פעולות:** אל תשנו הגדרות מסוכנות בשידור; רק הצביעו על המיקום.
- **קריינות:** "Command Palette… הגדרות הרחבה… היסטוריית שיחות, מספר סשנים, פקודות slash… לא חייבים הכול היום."
- **אם לא מספיק:** פתחו את הפלטה, חפשו מילה אחת ("Claude") והסבירו שכאן חוזרים כשהממשק משתנה בין גרסאות.

### S08 | 07:05–08:25 | גשר למתקדם

- **מה על המסך:** שקופית או מסך עם רשימה: worktrees, MCP, review, deploy; או כותרת הקורס המתקדם.
- **קריינות:** "במסלול המתקדם נעמיק ב-Cursor… כאן בניתם בסיס עם **Claude Code**…"
- **אם לא מספיק:** קיצרו ל-45 שניות — דברו על שני נושאים בלבד (worktrees + MCP).

### S09 | 08:25–09:30 | סיום ו-B03

- **מה על המסך:** כותרת B03 או מסוף ריק עם חץ.
- **קריינות:** "בשיעור הבא פקודת מסוף אחת: נשאל את קלוד להסביר לפני שמריצים — בלי פאניקה."
- **אם לא מספיק:** רק כותרת B03 + משפט אחד על "שאלה לפני Enter"; דלגו על הדגמת מסוף.
