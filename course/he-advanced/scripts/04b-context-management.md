## Reference Sections

### R01 | Context Budget

- לשאול תמיד: מהו המינימום שצריך כדי לפתור את המשימה הזאת?
- להבחין בין `stable project context` לבין `task-specific evidence`.
- לא לצרף היסטוריה ישנה אם אפשר לסכם אותה בשלוש עד חמש נקודות.

### R02 | Fresh Thread Handoff

- כש-thread ארוך מדי, לייצר handoff קצר:
- מה המטרה
- מה כבר נוסה
- אילו קבצים רלוונטיים
- מה עדיין חסר או לא ברור

### R03 | Output Shape Controls Cost

- לבקש `plan קצר`, `3 hypotheses`, `minimal diff`, או `stop after analysis`.
- לא לבקש "תסביר הכול" אם מה שצריך הוא החלטה אחת או צעד אחד.

## Prompt Asset

`Before solving this repository task, tell me the minimum useful context. Separate stable project context from task-specific context, ask for any missing evidence, and suggest a concise output format that fits the next decision. If the thread is too long, generate a 3 to 5 bullet handoff summary for a fresh conversation.`
