סיור מודרך בממשק — Agent, בורר המצבים, בורר המודל, Cursor Settings, ושינוי מקומי דרך בחירת קוד ו־`Cmd+I`. אין slides. שיעור זה הוא **צילום מסך חי לחלוטין** — כל ביט הוא הדגמה פעילה בתוך Cursor.

טיימקודים ליעדי עריכה; לכוונן אחרי picture lock.

---

## הכנה לפני ההקלטה

**צרו קובץ `index.html` בתוך `My-First-Project` עם התוכן הבא:**

```html
<!DOCTYPE html>
<html lang="he" dir="rtl">
<head>
  <meta charset="UTF-8">
  <title>My Project</title>
</head>
<body>
  <h1>כותרת ראשית</h1>
  <p>פסקה לדוגמה</p>
  <button>לחץ עלי</button>
</body>
</html>
```

קובץ זה הוא ה"בד" לכל ההדגמות בשיעור. הכניסו אותו לפרויקט, פתחו אותו ב-Editor, ועשו **git commit** לפני ההקלטה.

## 05:15 – 06:30 · שינוי מקומי עם בחירת קוד ו־Cmd+I

**מסך:** `index.html` פתוח. מסמנים עם העכבר את שורת ה־`<h1>` או את הטקסט שבתוכה.

**Overlay:**

```text
Select code
Cmd+I / Ctrl+I
שינוי מקומי דרך Agent
```

**פעולה:**

1. מסמנים את קטע הקוד הרלוונטי.
2. לחיצה על `Cmd+I` במק או `Ctrl+I` בווינדוס/לינוקס.
3. פאנל הסוכן נפתח עם ההפניה לקטע שסומן.
4. **זום** על ההפניה / reference שמופיעה בסוכן. חשוב לראות שהקטע המסומן צורף כהקשר.
5. הקלדה: `שנה רק את הכותרת הזאת ל"הפרויקט הראשון שלי"`
6. Enter.
7. **זום** על ה־diff או על ההצעה.
8. לחיצה על `Accept`.

**מסר ויזואלי:** השיעור צריך להמחיש שהבחירה בקוד אומרת לסוכן "תתמקד כאן".

## הערות הפקה כלליות

- **אין slides** בשיעור הזה — צילום מסך חי בלבד לאורך כל הדרך.
- **index.html מוכן מראש** — ראו הכנה בתחילת המסמך. אל תיצרו אותו בשידור חי בשיעור.
- **שמות ממשק:** השתמשו בשם שמופיע בגרסה שלכם בפועל.
- **Agent / בורר המודל / בורר המצבים** עשויים להיראות מעט שונה בין גרסאות. צלמו את מה שקיים באמת, לא מה שהיה בגרסה ישנה.
- בהדגמת השינוי המקומי, ודאו שרואים בבירור את ההפניה לקטע המסומן בתוך הסוכן.
- **פרטיות:** אין מידע אישי בשיעור — אין צורך בטשטוש.
- עשו **git commit** בסוף ההקלטה (מחוץ לצילום) אם נוספו שינויים לפרויקט הדמו.
