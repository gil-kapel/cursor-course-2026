# Lesson 1.6 — Chat modes: Ask, Agent, Plan, Debug

Hebrew-first lesson assets; this index is in English to match the course README.

**Scope:** One **real example** end-to-end: build a **Contact List** app (React + Tailwind CSS, local state, add/delete contacts). The lesson starts by **building the project with Agent**, then uses **Ask** to identify improvements, **Plan** to design them (including an explicit **plan revision** — e.g. add field validation + undo on delete), **Agent** to implement until verified **in the browser**, and **Debug** to fix a deliberate break with evidence. Short **Max mode** aside (context, cost, noise). Mode picker UI. **Canonical prompt text** lives in `on-screen.md` only.

## Files in this folder

## Syllabus topic

Build → Ask → Plan (including updating the plan) → Agent → validate in browser → Debug → optional Max mode context; mode switching and what to verify in each mode; one continuous demo instead of four disconnected mini-lessons. Bridges to lesson 1.7 (`Cursor Settings` and setup prompts).
